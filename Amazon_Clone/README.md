Amazon Clone
This project is a fully functional e-commerce front-end built entirely with HTML and CSS, replicating the familiar look and feel of Amazon’s website. Users can browse through products, explore categories, and experience an intuitive, responsive design—without any JavaScript.

Features
Responsive layout for various screen sizes

Intuitive navigation with Amazon-style menus and sections

Clean and consistent design for a seamless shopping experience

Project Structure
HTML
Defines the structure of the webpage, including:

Header – Amazon logo, delivery address, search bar, language selection, sign-in info, and cart icon.

Bottom Navigation Panel – Links to sections like Today's Deals, Customer Service, Registry, Gift Cards, and Sell.

Hero Section – Promotional message with a link to Amazon India.

Shop Section – Product categories with images and “See More” links.

Footer – Multiple link sections with information about Amazon.

CSS
Handles the visual styling for all webpage elements, including:

Header, navigation, and search bar

Bottom panel styling

Hero section background and text formatting

Shop section layout and hover effects

Footer layout and typography
